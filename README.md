Landing Page oficial para a capacitação em Inteligência Artificial dos servidores da Secretaria de Estado da Fazenda de Santa Catarina.

Esta página foi desenvolvida para servir como o portal de entrada e conversão da Jornada de IA, uma parceria estratégica entre a SEF/SC, Alura e FIAP. O objetivo é capacitar até 1.000 servidores em ferramentas de IA Generativa e análise de dados.

🎨 System Design & Identidade
O design foi construído seguindo os princípios de UI/UX modernos, utilizando o ecossistema visual da Alura e referências de interfaces de alta performance (como o GitHub).

Cores Primárias: Azul Alura (#2800ff) e Roxo de Destaque (#7a73ff).

Tipografia: Plus Jakarta Sans e Manrope (via Google Fonts).

Efeitos: Background animado com partículas (efeito GitHub) e scroll reveal dinâmico.

🏗️ Estrutura do Projeto
O site é composto pelas seguintes seções estratégicas:

Hero Section: Chamada de alto impacto com gatilho de inovação.

Trilhas de Aprendizado: Divisão clara entre o Nível 1 (Consciente) e o Nível 2 (Emergente).

Ecossistema: Prova social e autoridade através dos logos das instituições parceiras.

Calendário: Cronograma visual das aulas ao vivo (Maio/2026).

FAQ: Esclarecimento de dúvidas comuns para reduzir a fricção na inscrição.

🛠️ Tecnologias Utilizadas
Para garantir leveza e facilidade de manutenção, utilizamos:

HTML5 (Estrutura semântica)

Tailwind CSS (Estilização via CDN para prototipagem rápida)

JavaScript (Vanilla) (Animação de partículas e efeitos de scroll)

Google Material Symbols (Ícones modernos e leves)

🚀 Como Visualizar
Como este é um projeto front-end estático, basta abrir o arquivo index.html em qualquer navegador moderno.

Para Deploy Rápido:

Faça o upload deste repositório para o GitHub Pages.

O link estará disponível em minutos para compartilhamento com as diretorias da SEF.
